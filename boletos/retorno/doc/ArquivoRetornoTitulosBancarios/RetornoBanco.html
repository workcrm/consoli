Class RetornoBanco

Description

Classe que implementa o design pattern Strategy,

para leitura de arquivos de retorno de cobranças dos bancos brasileiros, vincular uma classe para processamento de uma carteira específica de arquivo de retorno, e criando uma interface única para a execução do processamento do arquivo.
